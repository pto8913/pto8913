素数判定

2019/09/28

概要

約数を列挙する

計算量

* O(√n)

実装例

python 
      
        def isprime(n):
          if n == 1:
            return False
          for i in range(2, int(n**0.5) + 1):
            if (n % i == 0):
              return False
          return True
C++ 
      
        template<typename T>
        bool isprime(T n) {
          if (n == 1) return false;
          for (int i = 2;  i * i <= n; ++i) {
            if (n % i == 0) return false;
          }
          return true;
        }
