最大公約数最小公倍数

2019/09/29

概要

最大公約数 gcdと最小公倍数 lcmを求める

計算量

* O(logN)

実装例

python 
      
        def gcd(n, m):
          while m:
            n, m = m, n%m
          return n
      
        def lcm(n, m):
          return n * m // gcd(n, m)
