特徴

会話、もしくは`TalkGraph`を終了します。

ノードアクション

PtoTalkNode_End固有のアクション

固有のアクションはありません。

変数

PtoTalkNode_End固有の変数

EndTalkType

  • `EndTalk` : 会話を終了します。
  • `OnlyThisGraph` : `TalkGraph`を終了します。
