概要

大別すると`Proxy`,`CustomEvent`,`Condition`,`End`,`JumpToProxy`ノードがあります。
もし独自のノードを追加したい場合はこれらのノードから派生してください。

ノードアクション

ノードによっては一部機能が制限されます、以下の画像のようなアクションが行えます。

PtoTalkNode固有のアクション

SelectChildNodes

そのノードの子ノードを一括で選択できます。
ノードがループする場合はノードのインデックスがそのノードより小さいノードを選択します。

Add Transition EnterEvent

このノードが実行される前に呼び出されるイベントを追加します。

Add Transition LeaveEvent

このノードが実行された後に呼び出されるイベントを追加します。

PtoTalkNode特有の変数

FPtoTalkNodeLinkedFunctionInfos

  • `Name` : 名前であり、タグでもあります
  • `Type` :
    • `ThisGraph` : このノードが存在する`TalkGraph`内に処理を遷移します。
    • `ParentGraph` : このノードが`PtoTalkNode_RunSubTalkGraph`ノードから実行されている場合に、`PtoTalkNode_RunSubTalkGraph`ノードを実行中の`TalkGraph`に処理を遷移します。
