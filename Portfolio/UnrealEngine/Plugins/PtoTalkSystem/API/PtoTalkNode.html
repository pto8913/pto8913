概要

大別するとProxy、CustomEvent、Condition、End、JumpToProxyノードがあります。
もし独自のノードを追加したい場合はこれらのノードから派生してください。

ノードアクション

ノードによっては一部機能が制限されます、以下の画像のようなアクションが行えます。

PtoTalkNode固有のアクション

SelectChildNodes

そのノードの子ノードを一括で選択できます。
ノードがループする場合はノードのインデックスがそのノードより小さいノードを選択します。

Add Transition EnterEvent

このノードが実行される前に呼び出されるイベントを追加します。

Add Transition LeaveEvent

このノードが実行された後に呼び出されるイベントを追加します。

PtoTalkNode特有の変数

FPtoTalkNodeLinkedFunctionInfos

  • Name : 名前であり、タグでもあります
  • Type :
    • ThisGraph : このノードが存在するTalkGraph内に処理を遷移します。
    • ParentGraph : このノードがPtoTalkNode_RunSubTalkGraphノードから実行されている場合に、PtoTalkNode_RunSubTalkGraphノードを実行中のTalkGraphに処理を遷移します。
