特徴

TalkGraph内部で別のTalkGraphを呼び出すためのノード

PtoTalkNode_CustomEventを継承したクラス。
そのため基本的にPtoTalkNode_CustomEventと同じ処理が行われますが、
ConvertToBPが行えません。


trueの場合、リンクされたイベントが終了後、Blueprintから次のノードを実行する必要があります。

ノードアクション

PtoTalkNode_RunSubTalkGraph固有のアクション

PtoTalkNode_CustomEventと同じなのでこちらをご覧ください。
なお、ConvertToBP、Add LinkedFunctionが行えません。
