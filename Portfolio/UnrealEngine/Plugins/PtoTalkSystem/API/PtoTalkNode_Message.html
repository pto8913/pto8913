特徴

  • `MessageRow`に設定された`DataTable`からテキストデータを読み取り、`TalkUI`上に表示します。
  • `WaitPlayerInput`が有効になっていると`PressToNext`ボタンが表示されます。
    • `MessageChoiceInfos`が有効になっている場合`WaitPlayerInput`は無視され、`ChoiceUI`が表示されます。
    • `ButtonText` : ボタンに表示されるテキスト。
    • `ChoiceType` : ボタンをクリックしたときにどのノードに遷移するかを選ぶためのデータ
    • `ChoiceTypeName` : カスタムできる`ChoiceType`
    • `OverrideButtonClass` : `ChoiceUI`に表示されるボタンのクラスを上書きします

ノードアクション

PtoTalkNode_Message固有のアクション

Add MessageFormatEvent

`MessageRow`を表示する前にテキストをフォーマットを行うことができます。
`MessageFormatEventInfos`に名前を設定する必要があります。
