特徴

  • MessageRowに設定されたDataTableからテキストデータを読み取り、TalkUI上に表示します。
  • WaitPlayerInputが有効になっているとPressToNextボタンが表示されます。
  • MessageChoiceInfosが有効になっている場合WaitPlayerInputは無視され、ChoiceUIが表示されます。
    • ButtonText : ボタンに表示されるテキスト。
    • ChoiceType : ボタンをクリックしたときにどのノードに遷移するかを選ぶためのデータ
    • ChoiceTypeName : カスタムできるChoiceType
    • OverrideButtonClass : ChoiceUIに表示されるボタンのクラスを上書きします

ノードアクション

PtoTalkNode_Message固有のアクション

Add MessageFormatEvent

MessageRowを表示する前にテキストのフォーマットを行うことができます。
MessageFormatEventInfosに名前を設定する必要があります。
※PtoTalkNode_Messageからのみ行えます。

PtoTalkNode_Message_Blueprintの場合はAdd MessageFormatEventはできません。
代わりにOnMessageFormatEventをOverrideすることができます。

Convert Native To BP

PtoTalkNode_MessageからPtoTalkNode_Message_Blueprintに変換することができます。
※PtoTalkNode_Messageからのみ行えます。

Convert BP To Native

PtoTalkNode_Message_BlueprintからPtoTalkNode_Messageに変換することができます。
※PtoTalkNode_Message_Blueprintからのみ行えます。
