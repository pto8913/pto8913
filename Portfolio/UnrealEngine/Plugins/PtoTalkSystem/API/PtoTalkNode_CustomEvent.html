特徴

`Add LinkedFunction`でリンクされたイベントを、実行するだけのノード
`ExecuteNextNodeManually`が`false`の場合、リンクされたイベントが終了時、次のノードに遷移します。

`true`の場合、リンクされたイベントが終了後、`Blueprint`から次のノードを実行する必要があります。
