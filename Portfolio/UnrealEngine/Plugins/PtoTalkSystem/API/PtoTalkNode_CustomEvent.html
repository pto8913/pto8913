特徴

Add LinkedFunctionでリンクされたイベントを、実行するだけのノード
ExecuteNextNodeManuallyがfalseの場合、リンクされたイベントが終了時、次のノードに遷移します。

trueの場合、リンクされたイベントが終了後、Blueprintから次のノードを実行する必要があります。
