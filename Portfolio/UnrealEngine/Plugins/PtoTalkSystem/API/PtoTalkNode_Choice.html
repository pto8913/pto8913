特徴

`PtoTalkNode_Condition`を継承したクラス。
そのため基本的に`PtoTalkNode_Condition`と同じ処理が行われますが、
`PtoTalkNode_Message`の`MessageChoiceInfos`と`ChoiceType`を一致させる必要があります。
なお、順番は関係ありません。

ノードアクション

`PtoTalkNode_Condition`と同じです。
