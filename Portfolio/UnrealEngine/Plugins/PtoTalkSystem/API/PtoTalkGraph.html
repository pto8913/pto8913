概要

次のページの`PtoTalkNode`を配置することができます。

アクション

各種LinkedFunctionの追加

Add LinkedFunction Default

`PtoTalkNode`の`Transient`や、`PtoTalkNode_CustomEvent`ノードの`LinkedFunction`にリンクすることができます。
`LinkedFunction`や`Transient EnterEvent`、`Transient LeaveEvent`に使用できます。

Add LinkedFunction Condition

`PtoTalkNode_Condition`や`PtoTalkNode_Choice`ノードの`LinkedFunction`にリンクすることができます。
※`Transient EnterEvent`や`Transient LeaveEvent`には使用できません。

Add LinkedFunction MessageFormatEvent

`PtoTalkNode_Message`ノードの`MessageFormatEvent`にリンクすることができます。

Export To CSV

`DataTable`を経由して`CSV`ファイルにエクスポートします。

    エクスポートされるデータ
  • 関数
  • 変数
  • PtoTalkGraphのノード
`DataTable`は`PtoTalkSystem/Content/EditorOnly/`に作成されます。
`CSV`ファイルは`PtoTalkSystem/Exports/`に作成されます。
`DataTable`に`ExportTableRowName`で設定された名前の行を追加します。
注意 ※`None`の場合は追加されません。
※すでに同じ名前の行がある場合はその行は削除されるので注意してください
※`CSV`ファイルは上書きされます。

Import From CSV

`CSV`ファイルを`DataTable`を経由してインポートします。
注意 ※`PtoTalkGraph`の内容はそのままに、ノードや関数、変数を作成するのでインポートする場合は空の`PtoTalkGraph`で行うことをおすすめします。
※`ReimportDataTable`が`true`の場合`DataTable`が`CSV`ファイルの内容で上書きされます。
