概要

次のページのPtoTalkNodeを配置することができます。

アクション

各種LinkedFunctionの追加

Add LinkedFunction Default

PtoTalkNodeのTransientや、PtoTalkNode_CustomEventノードのLinkedFunctionにリンクすることができます。
LinkedFunctionやTransient EnterEvent、Transient LeaveEventに使用できます。

Add LinkedFunction Condition

PtoTalkNode_ConditionやPtoTalkNode_ChoiceノードのLinkedFunctionにリンクすることができます。
※Transient EnterEventやTransient LeaveEventには使用できません。

Add LinkedFunction MessageFormatEvent

PtoTalkNode_MessageノードのMessageFormatEventにリンクすることができます。

Export To CSV

DataTableを経由してCSVファイルにエクスポートします。

    エクスポートされるデータ
  • イベントグラフ
  • 関数
  • 変数
  • PtoTalkGraphのノード
DataTableはPtoTalkSystem/Content/EditorOnly/に作成されます。
CSVファイルはPtoTalkSystem/Exports/に作成されます。
DataTableにExportTableRowNameで設定された名前の行を追加します。
注意
※Noneの場合は追加されません。
※すでに同じ名前の行がある場合はその行は削除されるので注意してください
※CSVファイルは上書きされます。

Import From CSV

CSVファイルをDataTableを経由してインポートします。
注意
※PtoTalkGraphの内容はそのままに、ノードや関数、変数を作成するので、インポートする場合は空のPtoTalkGraphで行うことをおすすめします。
※ReimportDataTableがtrueの場合DataTableがCSVファイルの内容で上書きされます。
