特徴

  • ProxyTagでPtoTalkNode_Proxyを判別します。
  • 基本的にPtoTalkNode_JumpToProxyからこのノードに遷移します。
  • TalkSystemCompからExecuteFromProxy関数を使って手動で呼び出すこともできます。

ノードアクション

PtoTalkNode_Startでは機能が制限されますが、以下の画像のようなアクションが行えます。

PtoTalkNode_Proxy固有のアクション

Jump To JumpToProxyNode

ProxyTagと一致するJumpToProxyInfos.Nameを持ったPtoTalkNode_JumpToProxyにジャンプします。

簡単な使用例

最初にHelloWorldのテキストが表示されます。
ボタンを押すとこのグラフのProxyTagがStartのPtoTalkNode_Proxyにジャンプします。
