特徴

`ProxyTag`で`PtoTalkNode_Proxy`を判別します。
基本的に`PtoTalkNode_JumpToProxy`からこのノードに遷移します。
また、`TalkSystemComp`から`ExecuteFromProxy`関数を使って手動で呼び出すこともできます。

簡単な使用例

最初に`HelloWorld`のテキストが表示されます。
ボタンを押すとこのグラフの`ProxyTag`が`Start`の`PtoTalkNode_Proxy`にジャンプします。
