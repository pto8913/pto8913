特徴

JumpToProxyInfos.Nameと一致するProxyTagと一致するPtoTalkNode_Proxyにジャンプします。

ノードアクション

PtoTalkNode_JumpToProxy固有のアクション

Add / Jump ProxyNode

JumpToProxyInfos.Nameと一致するProxyTagを持ったPtoTalkNode_Proxyノードを追加します。
もし、PtoTalkNode_Proxyノードが既に存在するなら、そのノードにジャンプします。

変数

PtoTalkNode_JumpToProxy固有の変数

JumpToProxyInfos

  • Name : 一致するProxyTagを探すために使用します。
  • Type :
    • ThisGraphの場合、同じTalkGraph内に存在するPtoTalkNode_Proxyに遷移します。
    • ParentGraphの場合、このノードが存在するTalkGraphをPtoTalkGraph_RunSubTalkGraphで、呼び出しているTalkGraph内に存在するPtoTalkNode_Proxyに遷移します。

簡単な使用例

PtoTalkNode_Proxyと同じなのでこちらをご覧ください。
